## URL Match Pattern

对于普通版, 不用对代码进行修改

对于国际版, 请将 `@match` 地址改成 `https://ticktick.com/`

## How to use

1. Install the [Tampermonkey](http://tampermonkey.net/) browser add-on.

2. Add the script `TickTickMarkdown.user.js` to Tampermonkey.

### Hotkey

F4 切换 Edit/Preview 模式.
